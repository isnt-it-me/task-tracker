# Task Tracker

A responsive Task Tracker web application built using the MERN Stack. This project demonstrates CRUD operations, REST APIs, MongoDB integration, form validation, filtering, sorting, and a modern responsive UI.

---

## Features

- Create, Update and Delete Tasks
- Search Tasks
- Filter by Status and Priority
- Sort Tasks
- Form Validation using React Hook Form and Zod
- Responsive Design
- Dark / Light Theme
- Toast Notifications
- RESTful API
- MongoDB Integration

---

## Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS
- React Router DOM
- React Hook Form
- Zod
- Axios
- React Toastify
- Lucide React

### Backend

- Node.js
- Express.js

- Mongoose

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/task-tracker.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm run dev
```

---

## Environment Variables

### Client (.env)

```env
VITE_API_URL=VITE_API_URL
```

### Server (.env)

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```
